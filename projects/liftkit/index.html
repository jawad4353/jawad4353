Liftkit™ (Matter Window Blind Controller)

Liftkit is a Flutter smart home app for controlling Matter-compatible window blind devices over the local network. Built on the Chip Controller SDK with a custom Matter fabric, it handles device provisioning, calibration, and open/close command delivery on the same LAN. Firebase powers authentication, with GetX for state management and certificate-based secure device communication.
